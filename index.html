Baseball Player Simulation

I (Peter Merrill) created Baseball Star in 2018 when my life was much simpler.

Baseball Star has been an amazing project, but I don't feel like I have enough time to properly manage the game anymore.

My wife and I had a baby last year and I recently got a new job. Both of these make it hard to find time to set aside for this personal project.

Thank you to everyone in the Baseball Star community who has played the game over the last few years. It has been a fun ride, but I am making the decision to shut the game down. I'd be open to making this open source if people were willing to pay the server costs ($10/mo) and maintain it. Let me know if you're interested in this.
